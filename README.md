## 项目名称
### Promise-code ![star](https://img.shields.io/github/stars/xuedongwang/promise-code?style=social)
## 项目背景
使用 `es6 class` 模拟 `Promise`，符合 `Promise+` 规范，完美通过所有单元测试。
## 使用
```
$ git clone https://github.com/xuedongwang/promise-code.git
$ npm i
$ npm run test
$ npm run lint
```
